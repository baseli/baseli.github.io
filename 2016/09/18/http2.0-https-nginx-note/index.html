nginx的https和http2.0踩坑记录

发表于 2016-09-18   |  

获取ssl证书

在开始配置服务器之前先需要获取ssl证书，这里选用的是沃通的免费证书，在这里进行简单的申请有一个好处就是，他会按照不同的服务器生成需要的证书。

开始安装nginx

由于需要使用https和http2.0，所以在开始的时候就需要安装openssl的库，而nginx又需要用到正则的库，所以还需要安装pcre库，在完成这两个库的安装之后就可以进行编译安装nginx了。

下载nginx源码包

解压，编译安装

等待上边的完成之后再执行make && make install完成安装，如果过程中有什么错误，可能是依赖包的关系，安装下就可以了。

配置nginx.conf

在/opt/nginx/conf/目录下有nginx.conf，用vim打开，然后进行配置:

  1. 开启https
  2. 开启http2.0
  3. 配置ssl证书相关
  4. 重启nginx服务
